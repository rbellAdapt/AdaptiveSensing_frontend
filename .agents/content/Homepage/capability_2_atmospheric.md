# Card Title
Atmospheric Sensing & Plume Modeling

# Card Description
Leveraging advanced analytical instrumentation—ranging from integrated mobile laboratories to UAS-mounted trace gas sensors—to conduct high-resolution, real-time chemical tracking. I specialize in Real-Time Data Acquisition System (RT-DAS) engineering and geospatial data fusion, capturing fugitive emissions (methane, VOCs) across extensive terrain.

These raw telemetry streams are ultimately synthesized into predictive atmospheric dispersion models.
